Начнем наш путь по классическому варианту - с определения что же такое CSS, для этого обратимся к книге Дэвида Макфарланда "Новая большая книга CSS":
Каскадные таблицы стилей, или Cascading Style Sheets (CSS), обеспечивают твор- ческую свободу в разметке и дизайне веб-страниц. Пользуясь CSS, вы сможете украсить текст страниц привлекательными заголовками, буквицами и рамками, как в красочных глянцевых журналах. Можно точно разместить и позиционировать изображения, сделать колонки и создать баннеры, выделить ссылки динамически- ми эффектами. Можно также добиться постепенного появления и исчезновения элементов, перемещения объектов по странице или медленного изменения цвета кнопки при прохождении над ней указателя мыши. CSS — это язык стилей. Он используется для того, чтобы придать страницам на HTML — фундаментальном языке Всемирной паутины — совершенный вид.
То есть CSS отвечает за красоту веб-страниц, тогда как HTML за её структуру и наполнение. Чтобы продемонстрировать всю мощь CSS предлагаю узреть данный сайт в его HTML обличии без каплии CSS.
И немного магии CSS в придачу.

Надеюсь магия вас впечатлила поэтому идем дальше: анатомия CSS. Любой стиль задается тремя обязательными параметрами - Селектором, Свойством и Значением. Селектор определяет к какому(каким) элементу будет применятся следующее свойство(свойства), в свою очередь Свойство определяет что именно мы хотим изменить: картинку на фоне, размер шрифта, положение объекта на странице или придать объекту анимацию. Значение определяет параметры своства: если мы выбрали размер шрифта то мы можем задать 25 пикселов, а можем просто поставить маленький размер шрифта - small. Структура написания стиля следующаяя:

Селектор{Свойство:Значение;}

Селекторы

Селекторы своего рода прицел для стиля который помогает выцелить необходимый элемент что бы применить к нему стиль. Существует несколько типов таких прицелов:


  • Селекторы тегов
  • Селекторы классов
  • Идентификаторы
  • Псевдоклассы
  • Псевдоэлементы

Селекторы тегов

Самые простые селекторы - селекторы тегов HTML, в данном случаем мы можем нацелить класс на все одинаковые теги в HTML коде, например селектор "a" применить стиль ко всем ссылкам, а селектор "header" к "шапке" сайта. Тут думаю в принципе ничего сложного, если вы знаете HTML то без труда справитесь.

Селекторы классов

Более продвинутые селекторы - селекторы классов, с помощью них можно уже более точечно применять стили, для этого необходимо в HTML-коде присвоить класс к любому тегу с помощью контрукции "<p class="one"> ", так мы присвоили класс "one" абзацу в документе, теперь чтобы нам применить стиль к данному азацу необходимо применить селектор ".one".

Идентификаторы

Идентификаторы похожи на классы, только в отличие от них немогут повторяться - каждый идентификатор должен быть уникален и не повторяться. Идентификаторы присваиваются элементам через параметр id ,например "id="headset"", обращение к элементу с таким id выглядит так "#headset". Есть и еще одно отличие идентификатора от класса - идентификатор имеет более высокий приоритет - если одному и томуже элементу задать два противоположных свойства, одно по классу, другое по идентификатору, то будет работать свойство заданое по идентификатору.

Псевдоклассы

Псевдоклассы определяют динамическое состояние элементов, которое изменяется с помощью действий пользователя, а также положение в дереве документа HTML. Например псевдокласс :hover оперделяет состояние элекмента при наведении на него указателя мыши, :first-child применяет стиль к первому дочернему элементу в родительском, :read-only применять стиль к элементам с атрибутом "только для чтения". Псевдоклассы довольно спецефичны для каждого элемента.
Применяются псевдоклассы в дополнения к другим селекторам, на пример к селекторам тегов "a:hover" или селекторам классов ".one:first-of-type".

Псевдоэлементы

Псевдоэлементы позволяют задать стиль элементов не определённых в дереве элементов документа, а также генерировать содержимое, которого нет в исходном коде текста. Псевдоэлементы начинаются с ::, чтобы отличить их от псевдоклассов.
Основных псевдоэлемента можно назвать два ::before и ::after, которые создают элемент соответсвенно до и после элемента к которому применяются. Например "p::after" добавляет элемент после абзаца.

Свойства и их значения

Свойств CSS огромное множество и рассмотреть все в данном формате не представлется возможным, поэтому я выделил четыре категории в рамках которых мы ознакомимся с основными свойствами.

  • Отступы и позиционирование
  • Размеры элементов
  • Свойства текста
  • Фон и цвет

Отступы и позиционирование

Одни из самых значимых свойств для макетирования страницы

  • Поля - margin
  • Отступы - padding
  • Позиционирование - position

Margin - свойство определяющее поля элемента со всех сторон, принимает значения в px, %, pt. Схожее с предыдущим свойство padding, в принципе они различаются только место применения margin устанавливает расстояние от других объектов, а padding расстояние содержимого документа от его границ.

Position - cложное вариативное свойство основные значения которого: absolute - абсолютное позиционирование, fixed - фиксированое позиционирование, relative - относительное позиционирование, static - статичное позиционирование, sticky - сочетание фиксированного и относительного позиционирование.

Размеры элементов

Совсем в кратце

  • Высота height
  • Ширина width

Данные свойства определяют размеры элемента и принимают в качестве значений px, %, em, in, pt.

Свойства текста

Для настройки тексового состоявляющих страницы существует несколько свойств.

  • Семейство шрифта - font-family
  • Размер шрифта - font-size
  • Межсимвольный интервал - letter-spacing
  • Межсловный интервал - word-spacing
  • Межстрочный интервал - line-height
  • Выравнивание - text-align

Семейство шрифта настраивается свойством font-family, в значение которого указывается семейство например "font-family: Geneva, Arial, Helvetica, sans-serif", здесь указано сразу 4 семейства шрифта, первый шрифт будет в приоритете, если его не окажется на компьютере то устанавливается следующий по порядку и так пока не дойдет до типа-шрифта:sans-serif, в таком случаем применется любой из шрифтов типа sans-serif который есть на компьютере.

Размер шрифта задается с помощью свойства font-size, которое принимает числовые значения в пикселах(px), процентах(%), а так же в em, epx, rem. Так же у данного свойства есть значения в виде ключевых слов:xx-small, x-small, small, medium, large, x-large, xx-large.
Например используя конструкцию "font-size:18px" мы выставляем размер шрифта 18 пикселов.

Межсимвольный интервал, Межсловный интервал, Межстрочный интервал - letter-spacing, word-spacing, line-height соответсвенно. Данные свойства согласно названием управляют расстоянием между буквами, словами, строками, они применяются со значениями в px, em, %.

Выравнивание текста выполняется свойством text-align, которое принимает следующие значени: left - выравнивание по левому краю, right - выравнивание по правому краю, center - выравнивание по центру, justify - выравнивание по ширине.

Фон и цвет

Ну и на последок разберем три свойства для придания сайту красок

  • Цвет текста - color
  • Цвет фона - background-color
  • Картинка фона - background-image

Цвет текста - color и цвет фона - background-color задаются одинаково, существует три способа задать цвет: написать цвет в текстовом виде(white, red, aquablue), написать кодировку цвета в RGB (#ff0000 - красный, #ff9900 -теплый оранжевый), задать в формате rgba (r, g , b, a) где первый три переменные значения цвета от 0 до 255, а последнее прозрачность от 0 до 1.

Для вставки картинки в качестве заднего фона применяется свойство background-image, которое в качестве значения принимает адресс картинки. Например background-image:url(example/images/bg.jpg).
